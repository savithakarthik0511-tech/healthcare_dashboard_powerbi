# healthcare_dashboard_powerbi

This project analyzes hospital appointment data using Power BI.

## Tools Used
- Power BI
- Excel
- DAX

## Key Insights
- Total Appointments: 1500
- No-show rate: 42.47%
- Highest patients are in age group 19-35
- SMS reminders reduced no-show rate
